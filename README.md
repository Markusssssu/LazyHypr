## Features
- Minimalist design with no visual clutter.
- Optimized for high-resolution (HiDPI) monitors.
- Fast and responsive workflow.

## Core Components
- **Window Manager:** Hyprland
- **Control Panel:** Hyprpanel
- **Terminal Emulator:** Kitty
- **App Launcher:** Wofi (Wayland fork)
- **Notification Management:** Dunst / Mako

## Appearance
This setup features a dark color palette with an emphasis on clean lines. It aims to provide a distraction-free environment while maintaining a modern aesthetic.

## Getting Started
Before proceeding with the installation, ensure your system is up to date and you have a working Wayland environment.

### Prerequisites
Make sure you have the following packages installed:
- `hyprland`
- `hyprpanel`
- `kitty`
- `wofi`

## Installation
To apply the **LazyHypr** configuration, clone the repository and run the setup script:

```bash
# Clone the repository
git clone git@github.com:Markusssssu/LazyHypr.git

# Enter the directory
cd LazyHypr

# Make the script executable and run it
chmod +x ./setup.sh
./setup.sh
